Antes de começar

Para utilizar os agentes, o usuário precisa ter uma conta no Claude IA. Esse requisito deve estar visível no hub para evitar dúvidas no acesso.

Requisito obrigatório É necessário possuir conta ativa no Claude IA. Sem esse acesso, os links dos agentes podem não funcionar corretamente para o docente.
